 Data Validation Tool


Description  This tool was created for the purpose of eliminating the regression testing in FSM Analytics or it can be used in other Birst Analytics projects. The user will just get the Birst Query Language that will be used for regression and will just paste it in the tool. You only need to validate the results in the first run and once it is verified, it can run every time and see if the results were passed or failed

How to Access

URL : http://phmadpenalo1/BirstTools

QA Tools>Data Validation Tool


 Model Validation Tool


Description  Automatically creates test scripts for each and every measure available from all model spaces defined, in FSM case, we have more than 5 model spaces. Each measure results are tested for “every” hierarchy and grain that is connected to it. The objective of the tool is to produce the same value even if there are hierarchies added on the measure.


How to Access

URL : http://phmadpenalo1/BirstTools

Development Tools>Model Validation Tool


 Package Repointing Tool


Description  For development use only, Birst have some issues that when we drop a package it losses all of its grains. With this tool, it will not drop the package hence it will just repoint the package from an old one to a newer one.

How to Access

URL :  http://phmadpenalo1/BirstTools

Development Tools>Package Repointing Tool

User Management Tool


Description  As Birst Analytics spaces are continuously increasing, it might be a tedious task to add multiple users in a space. This tool allows you to add multiple users at the same time by just putting a “;” between the user ids. Another cool feature is that you can just select spaces from the selection rather than going out and going in between spaces.


How to Access

URL : http://phmadpenalo1/BirstTools
